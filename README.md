Hermes科研助手 README文档
项目简介
Hermes科研助手是一款专为科研人员设计的辅助工具，基于Python开发，可通过自然语言指令触发核心功能，涵盖文献处理、实验辅助、科研器材指导三大核心场景，旨在降低科研操作成本，提升科研效率，充当科研人员的专属助手。
本项目配套完整的功能指令说明与可运行代码，支持文献添加与解析、实验路线制定、文献创新点提取、科研器材操作指导、实验计时记录等全流程科研辅助功能，适配各类实验类、综述类文献的处理需求。
核心功能
Hermes科研助手聚焦科研全流程，核心功能覆盖以下五大模块，所有功能可通过代码调用或自然语言指令触发（后续可拓展自然语言解析模块）：
1. 文献处理模块
- 文献添加：支持3种添加方式（全文粘贴、图片上传、DOI号检索），自动识别文献类型（实验类、综述类、理论类）；
- 文献解析：自动生成文献摘要、核心结论，快速提炼文献核心内容；
- 创新点提取：针对实验类文献，精准挖掘并分类创新点（实验方法创新、技术创新等），标注对应文献位置；
- 图片解读：提取文献中的图片（实验装置图、数据图等），结合文献上下文进行专业解读，保留图片关键细节。
2. 实验辅助模块
- 实验路线制定：结合实验类文献内容与现有实验条件，制定可落地、可调整的实验路线，包含步骤拆分、误差控制、路线优化建议；
- 实验计时记录：支持多任务计时，可设置计时时长、提醒节点，自动记录启动时间、结束时间，关联对应实验步骤，避免遗漏关键实验节点。
3. 科研器材指导模块
涵盖科研常用器材的规范操作指导，包括基础实验器材（烧杯、电子天平等）和精密仪器（离心机、PCR仪、分光光度计等），提供操作步骤、注意事项、常见故障排查方案，可直接查询具体器材的使用方法。
4. 疑问响应模块
针对科研过程中的各类疑问（文献添加失败、实验路线调整、计时错误、器材操作疑问等），提供精准、可操作的解决方案，全程响应使用者需求。
5. 可拓展性
代码结构清晰，可对接真实文献数据库API、图片识别接口、多线程定时任务，适配不同科研领域（生物、化学、物理等）的个性化需求，可灵活拓展新增功能。
环境配置
1. 运行环境
- Python版本：3.8及以上（推荐3.9-3.11）；
- 操作系统：Windows、macOS、Linux均可运行，无系统限制。
2. 依赖库安装
项目依赖以下Python库，需提前安装，安装命令如下（复制到终端/命令行执行）：
python hermes_biochem.py
